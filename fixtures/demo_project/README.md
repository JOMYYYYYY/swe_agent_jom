# Demo Project

This is a tiny fixture project for SWE agent experiments.

The project intentionally contains a simple bug. The expected agent workflow is:

1. Inspect the project structure.
2. Read the failing tests.
3. Search for the relevant implementation.
4. Identify the bug.
5. Later, after write/patch tools exist, fix the bug and run tests.

## Task

The calculator tests are failing. Find the bug in the implementation and explain the fix.
